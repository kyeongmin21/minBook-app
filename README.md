# minBook - app
## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start
   ```

3. Start the web (앱을 웹에서 확인하고 싶을 때)
    ```
   npx expo start --web
   ```


## 추후
- Expo Dev Client : npx expo install expo-dev-client
